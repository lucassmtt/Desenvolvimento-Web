Principais formatações

Negrito / destaque

Nesta frase conseguimos usar o destaque com a tag <b>(não sendo semântica)

Nesta frase conseguimos usar o destaque com a tag <strong> (semanticamente)

Itálico / Ênfase

Para utilizar o (Itálico) usamos a tag <i> (não semântica)

Para utilizar o italico de forma semantica, usamos a tag <em >

Marcação de texto:

Podemos utilizar uma marcação, usando a tag <mark>, ainda podemos trocar de cor, usando os parâmetros:
<mark style="background-color: exemplo-de-cor"> Texto de exemplo </mark> ;
